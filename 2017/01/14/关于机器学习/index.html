Numpy：

  1. 常用的统计函数：
    1. np.sum()
      • 返回求和
    2. np.mean()
      • 返回平均值
    3. np.max()
      • 返回最大值
    4. np.min()
      • 返回最小值
    5. np.std()
      • 返回标准偏差
    6. np.var()
      • 返回方差
    7. np.cumsum()
      • 返回累加值
    8. np.cumprod()
      • 返回累乘积值
  2. Numpy的数值类型：
    1. bool
      • 布尔类型True或者False 用1来表示True
    2. inti
      • 其长度取决于平台的整数，一般是int32或者int64
    3. int8
      • 字节长度的整数，取值为：[-128,127]
    4. int16
      • 16位长度的整数，取值为：[-32768,32767]
    5. int32
      • 32位长度的整数，取值为：[-231，231，-1]（**代表多少次方）
    6. int64
      • 64位长度的整数，取值为：[-263,263，-1]（**代表多少次方）
    7. uint8
      • 8位无符号的整数，取值为：[0,255]
    8. uint16
      • 16位无符号的整数，取值为：[0,65535]
    9. uint32
      • 32位无符号的整数，取值为：[0,2**32 - 1]
    10. uint64
      • 64位无符号的整数，取值为：[0,2**64 - 1]
    11. float16
      • 16位半精度浮点数：1位符号位，5位指数，23位尾数
    12. float32
      • 32位半精度浮点数：1位符号位，8位指数，23位尾数
    13. float64 或者可以直接叫float
      • 64位半精度浮点数：1位符号位，11位指数，52位尾数
    14. complex64
      • 复数类型实部和虚部都是32位浮点数
    15. complex28或者complex
      • 复数类型实部和虚部都是64位浮点数
